# workspace-settings Specification

## Purpose

工作台飞书式设置窗：左栏分类 + 右栏内容；通用分类可配置主题与会话显示；未接通分类显式占位。

## Requirements

### Requirement: 飞书式工作台设置窗

工作台 MUST 从资料名片「设置」打开独立设置界面（非仅迷你 popover 内嵌一行开关）。界面 MUST 提供左侧分类导航与右侧内容区。

#### Scenario: 打开设置

- **WHEN** 已登录成员在资料名片点击设置
- **THEN** 打开设置窗并默认选中「通用」分类
- **AND** 可见左侧至少一个其他分类入口（可为占位）

### Requirement: 通用分类可配置外观与会话显示

「通用」分类 MUST 提供：主题模式（跟随系统、浅色、深色）、主题色选择、会话气泡显示模式（左对齐、左右分布）。变更 MUST 立即作用于当前工作台会话（主题立即换肤；气泡布局作用于消息页气泡排列）。

#### Scenario: 切换主题模式

- **WHEN** 成员在通用中选择深色
- **THEN** 工作台 color mode 变为深色

#### Scenario: 切换会话显示模式

- **WHEN** 成员选择「消息气泡左右分布」
- **THEN** 消息页己方/对方气泡按左右分布展示
- **WHEN** 成员选择「消息气泡左对齐」
- **THEN** 消息页气泡均按左对齐规则展示

### Requirement: 未接通分类显式占位

设置窗中尚未实现的分类 MUST 明确为占位（文案或空态），点击 MUST 有可见反馈或占位页，不得静默无响应。

#### Scenario: 打开占位分类

- **WHEN** 成员选择尚未实现的分类（如隐私）
- **THEN** 右侧展示占位说明或 toast「功能即将推出」
